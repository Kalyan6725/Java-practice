Employee Management System

This application is built with Angular 22 and Spring Boot, providing a modern solution for managing employee data. It features real-time updates using Angular Signals, responsive design with Bootstrap 5, and RESTful API integration.

Key Features:
  • 🚀 Angular Signals: Reactive state management without RxJS complexity
  • 💼 CRUD Operations: Complete employee management functionality
  • 🎨 Bootstrap 5: Modern, responsive UI design
  • 🔄 Auto-refresh: UI updates automatically without page reload
  • 📱 Mobile-friendly: Works seamlessly on all devices
